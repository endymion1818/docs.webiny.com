## Create Content Model
1. From the **Side Menu**, Click **Headless CMS** > **Models**.  
2. Click **+ NEW MODEL**.
   
   ✔️ The New Content Model screen opens
3. Enter the Name and Description 
4. In the **Content model group** drop-down, select the group.
5. Click **+ CREATE MODEL**

   ✔️ Screen to configure content model's fields opens

## Define fields
1. Drag and Drop the **TEXT** field in the **EDIT** Tab.

   ✔️ Field Settings screen opens
2. Enter the field details.
   Note: You can add various validation to the field from **VALIDATORS** tab
3. Click **SAVE FIELD**

   ✔️ Field Settings screen will be closed
4. Click **SAVE**
